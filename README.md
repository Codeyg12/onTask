  ### Installation

To install follow these steps.

1.  Clone the repo using either HTTPS

```
https://github.com/Codeyg12/onTask.git
```

or SSH key

```
git@github.com:Codeyg12/onTask.git
```

2.  install NPM packages

`npm i`

3.  Start the app

`npm run develop`
